DialogTextContextMenuRepresentation

class DialogTextContextMenuRepresentation : TextContextMenuRepresentation

Dialog implementation of TextContextMenuRepresentation that provides minimal structure.

Unlike DefaultTextContextMenuRepresentation, this implementation doesn't provide any UI container. It only handles the context menu state and provides the onDismissRequest lambda to the content. This allows developers to create a completely custom UI for their context menus, such as dialogs or other complex components.
